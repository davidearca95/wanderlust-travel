# 🌍 Wanderlust - Travel Planner

App personale per pianificare viaggi con itinerari giorno per giorno.

## Features

- **Home con tile accattivanti** - Griglia di destinazioni con foto immersive
- **Ricerca** - Cerca per città, paese o tema
- **Periodo consigliato** - Selezione mesi con indicazione del periodo migliore
- **Itinerario dettagliato** - Giorno per giorno con timeline, orari, costi
- **Info pratiche** - Precauzioni, abbigliamento, prenotazioni, voli, valuta
- **Gallery** - Foto per ogni destinazione

## Destinazioni incluse

🇯🇵 Tokyo | 🇬🇧 Londra | 🇮🇩 Bali | 🇺🇸 New York | 🇮🇸 Islanda | 🇲🇦 Marrakech

## Setup

```bash
npm install
npx expo start --ios
```
